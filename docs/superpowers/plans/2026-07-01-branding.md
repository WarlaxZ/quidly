# Branding: "Quidly" Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebrand the app as **Quidly** — wordmark, favicon, metadata, and the "Self-Assessment, sorted." tagline — across the shell, login, and browser chrome.

**Architecture:** Pure presentation: swap the "Property Accounts" wordmark strings to "Quidly", update root metadata, add an `app/icon.svg` favicon (and remove the default `favicon.ico`), and add the tagline on login. No logic/routes/data touched; the suite stays green.

**Tech Stack:** Next.js 16 App Router (metadata + file-based `icon.svg`), the existing Fraunces wordmark styling.
